# content

## Purpose
Extension UI content directory containing bundled scripts and icons. These files are loaded directly by Zotero's browser engine.

## Key Files
| File | Description |
|------|-------------|
| `scripts/zotero-ner.js` | Main extension script (source) |
| `scripts/zotero-ner-bundled.js` | Webpack-bundled version of all modules |
| `scripts/zotero-ner-bundled.js.LICENSE.txt` | Bundle license information |
| `scripts/normalization-dialog-controller.js` | Dialog controller script |
| `scripts/ner-normalization-dialog.js` | Normalization dialog script |
| `icons/icon.svg` | Extension icon |

## Subdirectories
| Directory | Purpose |
|-----------|---------|
| `icons/` | Extension UI icons (SVG format) |
| `scripts/` | Scripts loaded into Zotero's content context |

## For AI Agents

### Working In This Directory
- `zotero-ner-bundled.js` is auto-generated by webpack from `src/index.js`
- Never edit `zotero-ner-bundled.js` directly - edit source in `src/` and rebuild
- Icon is displayed in Zotero's extension UI

### Testing Requirements
- Bundled file should match source files after rebuild
- Verify icon renders correctly in Zotero

### Common Patterns
- UMD module format for browser compatibility
- SVG icons scale to any size

## Dependencies

### Internal
- `src/` - Source files bundled by webpack

### External
- None (content files are bundled output)
